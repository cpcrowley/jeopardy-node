How to Use

Predefined Queries Tab

  1. Select a question from the dropdown menu
  2. Choose a season range to analyze (e.g., seasons 35-42 for recent games)
  3. Click Analyze to run the query

Ask Question Tab

  1. Enter a natural language question about Jeopardy statistics
  2. Add an optional summary (~5 words) for easy reference
  3. Choose a season range to analyze
  4. Click Ask - Claude AI will generate and execute analysis code

Your questions are saved automatically for future use.

About the Data

This app analyzes historical Jeopardy! game data scraped from j-archive.com. Each game includes scores at the end of each round, Final Jeopardy wagers, and which contestants answered each clue correctly.
